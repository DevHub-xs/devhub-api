# Changelog

All notable changes to DevHub API will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2024-11-09

### Added
- Proprietary LICENSE (All Rights Reserved)
- Professional README with badges and concise documentation
- Enhanced SECURITY.md with CodeQL scan results
- CONTRIBUTING.md with development workflow
- CODE_OF_CONDUCT.md for professional collaboration
- GitHub issue templates (bug report, feature request)
- Pull request template
- .gitattributes for consistent file handling
- .editorconfig for coding standards
- Enhanced package.json metadata

### Changed
- Updated README to be more concise and professional
- Enhanced SECURITY.md with vulnerability reporting
- Changed license from MIT to UNLICENSED (proprietary)
- Version bump to 0.1.0

## [0.0.1] - 2024-11-08

### Added
- Initial microservices architecture
- User management service with JWT authentication
- Service catalog management
- Developer tools integration
- MongoDB database integration with Mongoose
- Express.js REST API
- Docker and Docker Compose support
- Rate limiting middleware
- Input validation with express-validator
- Security headers with Helmet
- CORS configuration
- Comprehensive error handling
- Role-based access control (RBAC)
- Password hashing with bcrypt

### Security
- Implemented JWT authentication
- Added rate limiting (API, Auth, Registration)
- Input validation on all endpoints
- Password hashing with bcrypt (10 rounds)
- Security headers with Helmet
- CORS protection
